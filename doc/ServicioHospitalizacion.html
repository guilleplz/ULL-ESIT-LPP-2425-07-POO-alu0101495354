    class ServicioHospitalizacion

    Clase que representa un servicio de hospitalización, derivado de ServicioSalud.

    Esta clase extiende de `ServicioSalud` y añade funcionalidad específica para los servicios de hospitalización, incluyendo el número de plantas disponibles y un método para calcular las camas libres en el servicio de hospitalización.

    @see ServicioSalud
