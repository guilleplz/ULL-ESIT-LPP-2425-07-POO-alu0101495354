    class ServicioUrgencias

    Clase que representa un servicio de urgencias, derivado de ServicioSalud.

    Esta clase extiende la funcionalidad de `ServicioSalud` añadiendo un atributo específico para gestionar las camas UCI disponibles en el servicio de urgencias. También implementa métodos para gestionar las camas libres, considerando tanto las camas estándar como las camas UCI.

    @see ServicioSalud

    numero_camas_libres() click to toggle source

    Calcula el número total de camas libres en el servicio de urgencias.

    Este método suma el número de camas libres tanto en la unidad de cuidados intensivos (UCI) como en las camas estándar. Las camas estándar se consideran libres si tienen el valor `nil`.

    @return [Integer] Número total de camas libres en el servicio de urgencias.

def numero_camas_libres

end
